# Boilerplate FastAPI v2

### Following real world application boilerplate

###### Setup and run .venv

```bash
python -m venv .venv
```

```bash
source .venv/bin/activate # macOS

.venv\Scripts\activate #windows
```

###### Install dependencies

```bash
pip install -r requirements.txd
```

###### Run the app

```bash
uvicorn main:app --reload
```
